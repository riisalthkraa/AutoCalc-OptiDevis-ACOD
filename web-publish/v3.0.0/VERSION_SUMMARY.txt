    🐛 BUGS CORRIGÉS
═══════════════════════════════════════════════════════════════════════

[CRITIQUE] IDs dupliqués formulaire devis
   • Symptôme: Email, téléphone, adresse ne se sauvegardaient pas
   • Cause: #client-email existait 2 fois dans index.html
   • Fix: Renommage en #client-form-email dans modal client
   • Fichiers: index.html, ClientsModule.js

[CRITIQUE] DataCache.remove() inexistante
   • Symptôme: Erreur à la restauration de backup
   • Cause: Méthode legacy supprimée
   • Fix: Remplacement par DataCache.invalidate()
   • Fichiers: CacheKeys.js

[MAJEUR] Champs vides en édition client
   • Symptôme: Champs vides malgré données existantes
   • Cause: Conflit d'ID
   • Fix: Résolu par correction précédente
   • Impact: 100% des modifications clients fonctionnent
