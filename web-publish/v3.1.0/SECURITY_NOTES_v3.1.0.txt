  RECOMMANDATIONS UTILISATEURS
═══════════════════════════════════════════════════════════════════

🔐 Bonnes pratiques:

1. MOTS DE PASSE
   → Utilisez un mot de passe fort (12+ caractères)
   → Mélangez majuscules, minuscules, chiffres, symboles
   → Ne partagez jamais vos identifiants
   → Changez le mot de passe par défaut "admin123"

2. BACKUPS
   → Effectuez des backups réguliers
   → Stockez les backups dans un emplacement sécurisé
   → Testez vos restaurations
   → Gardez plusieurs versions

3. SYSTÈME
   → Maintenez Windows à jour
   → Utilisez un antivirus
   → Activez le pare-feu Windows
   → Évitez les clés USB inconnues

4. DONNÉES SENSIBLES
   → Ne stockez pas de données bancaires
   → Soyez prudent avec les informations personnelles
   → Respectez la confidentialité de vos clients
   → Documentez vos accès
